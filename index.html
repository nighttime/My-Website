Echo
What makes a shareable idea?
Echo is a an experiment in meme propagation. It models the way an idea spreads through a population using a one-way message system in order to explore the quirks and whimsies that make up popular ideas.
It starts with a user sharing a message on Echo. It's read by a random user around the world. If they share the message, too, it's read by another user and can be shared again. However, if it isn't passed on, the message chain ends and no one will see the message again.
A message is more popular the more times it has been echoed. All it takes is a single user to break the chain though, and the message will be muted.
Echo is an iOS project with a web companion. I conceived of the idea, directed all project-wide style, and designed and engineered the mobile UI.

Lightning

Notes

Lightning Notes is a note-taking web app. It understands the way you take notes, and can style and structure text as you type.
Lightning translates rules you define into regular expressions which it uses to detect patterns in your notes for styling.
A style rule can describe font use for keywords or for text after a keyword, horizonatal alignment, background and text coloring, and more.
I developed the product idea for Lightning, led interface design efforts, and engineered the text comprehension engine, which includes the process of dynamically compiling user rules into regular expressions.
